UPI Adoption & Cashless Economy – Data Analysis Project

A complete end-to-end data analysis study to understand how India is adopting UPI, why users trust it, what issues they face, and why some still hesitate to go digital.
Survey-based + KPI Dashboard + Real Insights


About the Study:-

Stream: SY B.Sc. Data Science

Method: Google Form survey

Sample Size: 60 valid respondents

Focus Areas:
✔ UPI usage adoption
✔ Most used UPI apps
✔ Benefits & issues
✔ Security perception
✔ Non-user reasons
✔ Youth involvement


Tools & Techniques Used:-

Microsoft Excel:
▪ Power Query (Data Cleaning)
▪ PivotTables (Analysis)
▪ KPIs + Charts (Dashboard)

Survey Design & Data Collection

Data Visualization & Reporting


Key Insights (Dashboard Results):-

| Insight             | Result                         |
| ------------------- | ------------------------------ |
| UPI Adoption        | 93% active users               |
| Leading App         | Google Pay — 52%               |
| Youth Usage (18–25) | Highest – 72%                  |
| Top Benefit         | Instant & Easy                 |
| Main Problem        | Payment delays & security fear |
| Satisfaction        | 74% satisfied                  |
| Security Trust      | 83% believe UPI is secure      |

Insights derived from cleaned dataset using Power Query & visual dashboards
